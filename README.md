#### - 간단 소개
> `NEO`는 `Need Each Other`로, 스타와 팬은 서로를 필요로 하는 관계에서 착안해 제작중인 서비스입니다. 스타의 일상을 공유하고, 자유와 제한을 통해 성숙한 소통을 할 수 있는 `스타페이지`, 팬들에게 질문해 가장 많이 공감하는 말을 들을 수 있는 `스타톡`, `암표걱정 없는 콘서트/팬미팅 예매 기능`을 목표로 제작중입니다.
